# Python运算符

1. `+`：加法运算符，将两个数相加并返回结果。
2. `-`：减法运算符，将两个数相减并返回结果。
3. `*`：乘法运算符，将两个数相乘并返回结果。
4. `/`：除法运算符，将两个数相除并返回结果，结果为浮点数。
5. `//`：整除运算符，将两个数相除并返回结果，结果为整数，向下取整。
6. `%`：取模运算符，返回两个数相除的余数。
7. `**`：幂运算符，将一个数的值乘以另一个数次幂的值，并返回结果。

```python
# 加法运算
print(2 + 3)  # 5

# 减法运算
print(5 - 2)  # 3

# 乘法运算
print(2 * 3)  # 6

# 普通除法
print(5 / 2)  # 2.5

# 整除运算
print(5 // 2)  # 2

# 取模运算
print(5 % 2)  # 1

# 幂运算
print(2 ** 3)  # 8

```
